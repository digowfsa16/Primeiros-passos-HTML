Um resumo sobre HTML

fonte: ia do google
"HTML (Linguagem de Marcação de Hipertexto) é uma linguagem de computador que define a estrutura e o significado do conteúdo de uma página web. Ele é o bloco de construção básico da web e está presente na maioria das páginas da internet e dos aplicativos online. O HTML é utilizado para: Estruturar o conteúdo de uma página, como em parágrafos, listas, imagens e tabelas Organizar e formatar informações Criar links entre páginas e sites Formatar documentos de forma similar ao Microsoft Word O HTML não é uma linguagem de programação


O HTML foi criado pelo físico Tim Berners-Lee para a comunicação e disseminação de pesquisas entre ele e seus colegas de faculdade. A versão mais usada até hoje é a 4.1, lançada em 1999 pelo W3C, a comunidade que normatiza e desenvolve padrões para a internet 

Algumas das tags mais utilizadas são:
        <head> – Local para declarar informações, como título e metadados da página
        <title> – Define o título
        <body> – Local para declarar todos os elementos que irão compor o corpo da página
        <p> – Tag para definir um parágrafo
        <a> – Tag de link
        <table> – Define uma tabela
        <ol> – Define uma lista ordenada
        <ul> – Define uma lista não ordenada
        <form> – Define um formulário
        <img> – Permite inserir uma imagem no documento
Utilizando um seletor de cor para mudar a cor do titulo , pode ser trocado ao selecionar cor ou confirmando no botão
